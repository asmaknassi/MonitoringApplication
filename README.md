Réalisation d’une application de surveillance et de détection de mouvement

L’application prend une photo de la personne qui s’approche du capteur et qui dépasse la distance prédéfinie par le capteur ultrason 
en utilisant une connexion Bluetooth, un sms est envoyé contenant le message d’alarme, ainsi que les informations concernant la distance
seront stockées dans une base de données SQLITE.
Ensuite afficher sous forme d’un graphe la distance et le temps (en temps réel / depuis la base de données).
